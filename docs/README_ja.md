AirText は軽量な macOS メニューバーアプリケーションで、Mac 上のどこでも音声でテキストを入力できます。**Fn** キーを押しながら話すだけで、文字起こしされたテキストが現在フォーカスされている入力フィールドに自動的にペーストされます。

## ✨ 機能

- **🎤 押して話す** — Fn を押して録音、離すと自動で文字起こしとペースト
- **⚡ リアルタイムストリーミング** — 話しながらリアルタイムで文字起こしを表示
- **🌊 波形アニメーション** — リアルタイム音声ビジュアライゼーション付きのエレガントなフローティングパネル
- **🌍 5言語対応** — English、简体中文、繁體中文、日本語、한국어
- **🤖 LLM テキスト補正**（オプション）— OpenAI 互換 API で音声認識エラーを自動修正
- **📋 スマートペースト** — CJK 入力メソッドの切り替えとクリップボードの復元を自動処理
- **🪶 軽量** — メニューバーアプリ、Dock アイコンなし、最小限のリソース使用

## 📋 システム要件

- **macOS 14.0**（Sonoma）以降
- **Swift 5.9** 以降
- **Xcode コマンドラインツール**（ビルド用）

## 🚀 クイックスタート

### 1. リポジトリをクローン

```bash
git clone https://github.com/your-username/airtext-mac.git
cd airtext-mac
```

### 2. ビルド

```bash
# デバッグビルド（高速、初回実行におすすめ）
make debug

# リリースビルド（最適化済み）
make build
```

### 3. 実行

```bash
# アプリを実行
make run-debug
```

またはプロジェクトディレクトリ内の `AirText.app` をダブルクリックしてください。

### 4. アクセス許可の付与

初回起動時に、macOS が以下のアクセス許可を求めます：

| アクセス許可 | 用途 | 設定場所 |
|---|---|---|
| **マイク** | 音声録音 | システム設定 → プライバシーとセキュリティ → マイク |
| **音声認識** | 音声テキスト変換 | システム設定 → プライバシーとセキュリティ → 音声認識 |
| **アクセシビリティ** | グローバル Fn キー監視 | システム設定 → プライバシーとセキュリティ → アクセシビリティ |

> ⚠️ **重要**：AirText が正しく動作するためには、**3つすべてのアクセス許可**を付与する必要があります。アクセシビリティの許可を付与した後、アプリの再起動が必要な場合があります。

### 5. 使ってみましょう！

1. メニューバーの**波形アイコン**（🎵）を確認
2. 任意のアプリのテキストフィールドをクリック
3. **Fn を押す** — 波形アニメーション付きのフローティングパネルが表示
4. **話す** — リアルタイムで文字起こしが表示
5. **Fn を離す** — テキストが自動的にペースト

## 🛠️ インストール

AirText をアプリケーションフォルダにインストール：

```bash
make install
```

アンインストール：

```bash
make uninstall
```

## ⚙️ 設定

### 言語の切り替え

メニューバーアイコンをクリック → **Language** → 言語を選択：

- English (en-US)
- 简体中文 (zh-CN) — *デフォルト*
- 繁體中文 (zh-TW)
- 日本語 (ja-JP)
- 한국어 (ko-KR)

### LLM テキスト補正（オプション）

AirText はオプションの LLM ベースのテキスト修正機能をサポートしており、一般的な音声認識エラー（同音異字、技術用語など）を修正できます。

1. メニューバーアイコンをクリック → **LLM Refinement** → **Settings...**
2. API 情報を入力：
   - **Base URL**：OpenAI 互換 API アドレス（デフォルト：`https://api.openai.com/v1`）
   - **API Key**：API キー
   - **Model**：モデル名（デフォルト：`gpt-4o-mini`）
3. **Test Connection** をクリックして接続を確認
4. メニューバーアイコン → **LLM Refinement** → **Enable** で有効化

> 💡 OpenAI 互換の API であれば何でも使用可能です（OpenAI、Ollama、LM Studio など）

## 🔧 Make コマンド

| コマンド | 説明 |
|---|---|
| `make build` | リリース版をビルド |
| `make debug` | デバッグ版をビルド（高速） |
| `make run` | リリース版をビルドして実行 |
| `make run-debug` | デバッグ版をビルドして実行 |

| `make clean` | すべてのビルド成果物を削除 |
| `make help` | 利用可能なコマンドを表示 |

## 🔒 プライバシーとセキュリティ

- **データ収集なし** — すべての音声認識は Apple Speech フレームワークによりローカルで処理
- **ハードコードされた認証情報なし** — API キーはユーザーが入力しローカルに保存
- **LLM はオプション** — LLM を設定しなくても完全に動作
- **オープンソース** — 完全なソースコードをレビュー可能
